📖 系统介绍

这是一个基于 Spring Boot 的校园选课系统，提供完整的课程管理、学生管理和选课管理功能。

🔗 API 接口文档

GET 查询所有课程
/api/courses
获取系统中所有课程的列表
GET 查询单个课程
/api/courses/{id}
根据ID获取特定课程的详细信息
POST 创建课程
/api/courses
创建新的课程（需要JSON数据）
PUT 更新课程
/api/courses/{id}
更新指定课程的信息
DELETE 删除课程
/api/courses/{id}
删除指定的课程
GET 查询所有学生
/api/students
获取系统中所有学生的列表
POST 创建学生
/api/students
创建新的学生账户（自动生成ID和时间戳）
POST 学生选课
/api/enrollments
学生选择课程（需要courseId和studentId）
GET 查询选课记录
/api/enrollments
获取所有选课记录

📋 使用说明

1. 使用 Postman、Apifox 或 curl 测试 API 接口

2. 所有 API 返回统一的 JSON 格式

3. 系统使用内存存储，重启后数据会重置

4. 支持课程容量限制、重复选课检查等业务规则
